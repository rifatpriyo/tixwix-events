# TixWix

Your premier destination for booking movies, concerts, and MLS soccer tickets.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase (Auth, Database, Storage)

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Features

- 🎬 Movie ticket booking with seat selection
- 🎵 Concert ticket purchasing
- ⚽ MLS Soccer match tickets with stadium view
- 🔐 User authentication (email/password)
- 📱 Fully responsive design
